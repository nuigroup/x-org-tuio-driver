X-org-tuio-driver

X.org multitouch driver for TUIO trackers.

View the Project on GitHub nuigroup/x-org-tuio-driver

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

The goal of this project is to create an X.org multitouch driver for TUIO trackers. MPX + Xinput2 are not in a release version of X.org yet, so at the moment a custom build is required.

For more information go to my blog at (http://rhuffman.wordpress.com/)
